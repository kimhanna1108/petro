TRAXON Synthetic 75W-90은 API GL-5가 요구되는 하이포이드 액슬과, API MT-1이 요구되는 일부 비동기식 수동 변속기에서 극한 조건의 모든 계절에 걸쳐 궁극적인 보호를 제공합니다.


TRAXON Synthetic 75W-85는 에너지 효율이 뛰어난 액슬 오일로, 하이포이드 액슬에서 탁월하고 오래 지속되는 마모 보호를 제공하여 장비 수명을 연장하고 가동 중단 시간을 줄여줍니다. 이 제품은 API GL-5가 요구되는 곳과 API MT-1이 필요한 일부 비동기식 수동 변속기에 사용됩니다..


TRAXON Synthetic CD-50는 에너지 효율이 뛰어난 상용차 수동 변속기 오일로, Eaton, Mack, International/Navistar, Meritor 등에서 제조한 OEM 연장 보증 프로그램 적용 대상이 아니게 된 구형 중장비 수동 변속기에 사용하도록 권장됩니다.


TRAXON Synthetic MTF 75W-80은 프리미엄 성능의 에너지 효율이 높은 상용차 수동 변속기 오일/기어 오일로, 최신 동기화 자동 수동 변속기(Synchronized Automated Manual Transmissions)를 위해 특별히 제조되었습니다. 이 제품은 긴 교환 주기, 부품의 긴 수명, 최적의 기어 변속 성능을 제공합니다. Volvo I-Shift 및 Mack mDrive 동기화 자동 수동 변속기 애플리케이션에 적합합니다.

  • 모두 가장 까다로운 환경에서도 고온 및 극한의 추운 날씨에 연중 보호 기능을 제공합니다.
  • 탁월한 전단 안정성(shear stability)을 제공하여 점도 유지, 저온 보호 및 토크 효율을 돕습니다.
  • 모두 긴 교환 주기를 제공하여 교환 횟수를 줄이고 오일 수명을 극대화합니다.
