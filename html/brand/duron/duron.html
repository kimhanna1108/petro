왜 DURON을 선택해야 할까요?

엔진 보호 및 내구성

DURON은 API CK-4 및 FA-4의 새로운 규격 요구 사항을 충족하기 위해 개발된 가장 내구성이 뛰어난 오일입니다.


가장 혹독한 엔진 성능 테스트 중 하나로 널리 알려진 Cummins ISB 테스트 조건을 뛰어넘는 DURON은 엔진 수명을 연장하기 위해 높은 수준의 엔진 보호 성능을 제공합니다.


연장된 오일 교환 주기

차량과 장비가 오일 교환 주기 사이를 더 오래 운행할수록, 운영 효율은 높아지고 윤활유 비용은 줄어듭니다.


TDURON의 탁월한 내구성 덕분에 신뢰성이나 엔진 보호를 희생하지 않고도 교환 주기를 연장할 수 있습니다.[1]


극한 작업 환경

온도 극한은 윤활유 성능에 큰 도전 과제입니다. 매우 높은 온도는 일반 엔진 오일을 약화시키고 분해시키며, 매우 낮은 온도는 오일의 흐름을 방해합니다.


Petro-Canada 윤활유는 사계절 엔진오일 분야의 선두주자이며, DURON은 신뢰할 수 있는 성능을 제공합니다.


연료 효율성

도로 주행과 비포장 도로 주행 모두에서 연료 소비는 모든 중장비 사업에 중요한 비용입니다. API CK-4와 FA-4 규격 모두를 충족하는 DURON은 낮은 점도의 엔진 오일을 제공합니다.


이 오일들은 엔진 내 오일의 점성 저항을 줄여주어, 엔진이 더 효율적으로 작동할 수 있게 하며 연료 사용량을 줄일 가능성이 있습니다.
